# Hi, I'm Gaurav 👋

I'm a final-year IT student at Poornima College of Engineering (Batch 2027) who builds full-stack web applications and AI-powered tools. My work sits at the intersection of backend systems, automation, and developer tooling.

I work primarily with **React, Next.js, Node.js, TypeScript, and MongoDB**. Most of my recent projects involve integrating AI APIs into real products, not just prototypes.

I care about systems that hold up under real conditions: clear API contracts, predictable data flow, and backends that don't break when load hits. I think through architecture before touching code.

**Open to internships and full-time roles** — remote, Delhi NCR, or Jaipur.

## Projects

### [InboxPilot-AI](https://github.com/gauravjain0377/InboxPilot-AI)
Full-stack, AI-powered email management platform. Handles email triage, automated follow-up scheduling, and inbox prioritization. Built with Next.js, TypeScript, and integrated with AI APIs for classification and response drafting.

### [StockSathi](https://github.com/gauravjain0377/StockSathi)
A full-stack stock trading platform with buy/sell execution, holdings tracking, and real-time order history. Built with React, Node.js, and MongoDB. Focused on clean data modeling and consistent state across trading flows.

### [Devora](https://github.com/gauravjain0377/devora)
A hackathon management system built during my internship. Handles participant registration, team formation, project submission, and judging workflows. Includes live announcements, badge system, and chat rooms. MERN stack with faker-based seed data for testing.
