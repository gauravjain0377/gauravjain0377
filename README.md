## About

Final-year IT student at Poornima College of Engineering (Batch 2027). I build full-stack web applications and AI-powered tools — things that work under real conditions, not just in demos.

Most of my work involves React, Next.js, Node.js, TypeScript, and MongoDB. Recent projects lean into LLM integration: wiring Gemini and OpenAI APIs into products that handle real workflows, not just toy queries.

I think through architecture before writing code — clear API contracts, predictable data flow, databases that model the domain correctly.

**Open to internships and full-time roles** · Remote · Delhi NCR · Jaipur

## Projects

### [InboxPilot-AI](https://github.com/gauravjain0377/InboxPilot-AI)
AI-powered email management platform. Handles triage, automated follow-up scheduling, and inbox prioritization. Integrates LLM APIs for email classification and response drafting. Built with Next.js and TypeScript.

### [StockSathi](https://github.com/gauravjain0377/StockSathi)
Full-stack stock trading platform with buy/sell execution, holdings tracking, and real-time order history. React, Node.js, MongoDB. Built around clean data modeling and consistent state across trading flows.

### [HackZen](https://github.com/gauravjain0377/HackZen)
Hackathon management system built during my internship. Covers participant registration, team formation, project submission, and judging workflows. Includes live announcements, badge system, and in-app chat. MERN stack.

## Open Source

### [HumanWrites](https://github.com/gauravjain0377/humanwrites) · `npm install humanwrites`
Detects and removes 33 AI writing patterns from generated text. Based on Wikipedia's Signs of AI Writing guide with 8 original additions. Published on npm at v2.0.3.
